# 🌦️ Weather App

A stylish and responsive weather application built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time weather data, air quality, and 5-day forecasts using the **OpenWeatherMap API**.

---

## 🔥 Features

- 🌍 Get weather using **city name** or **current location**
- 🕑 Shows **current temperature**, **weather condition**, and **location**
- 📅 Displays **5-day forecast** with icons and temperature
- 🌤️ Shows **hourly forecast** for the current day
- 🌫️ Displays **Air Quality Index (AQI)** with pollutants like PM2.5, PM10, SO2, etc.
- 🌅 Shows **sunrise & sunset** times
- 💧 Shows **humidity, pressure, visibility, wind speed, and feels like temperature**

---

## ⚙️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
- **OpenWeatherMap API**
- **Font Awesome & Boxicons** for icons
- **Moment.js** for date formatting
